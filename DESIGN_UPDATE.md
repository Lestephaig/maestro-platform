# Обновление дизайна Maestro Platform

## Новая цветовая схема

### Основные цвета:
- **Черный**: `#0a0a0a` - основной фон
- **Темно-серый**: `#1a1a1a`, `#2a2a2a` - карточки и контейнеры
- **Белый**: `#ffffff` - основной текст
- **Золото**: `#d4af37` с градиентами (`#b8941f` → `#d4af37` → `#f0d878`)
- **Серебро**: `#c0c0c0` с градиентами (`#a8a8a8` → `#c0c0c0` → `#e8e8e8`)
- **Синий акцент**: `#4a90e2` (для календаря и акцентов)

## Ключевые особенности дизайна

### Металлические эффекты
```css
/* Золотой градиентный текст */
background: linear-gradient(135deg, var(--gold-light), var(--gold-dark));
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

/* Свечение */
text-shadow: 0 0 20px rgba(212, 175, 55, 0.5);
box-shadow: 0 10px 30px rgba(212, 175, 55, 0.4);
```

### Анимации
- Волновой эффект на кнопках при hover
- Shine-эффект на карточках
- Плавные transitions на всех интерактивных элементах
- Анимация появления dropdown меню

### Адаптивность
- Все страницы полностью адаптивны
- Mobile-first подход
- Оптимизированные отступы для мобильных устройств
